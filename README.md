# Porter - python port scanner
--Usage : 
<pre>
python porter.py <target> --- Scan most common ports (1000)
<pre/>

--Example :
<pre>
python porter.py 192.168.1.10
<pre/>

--Scan specific ports
<pre>
python porter.py <target> -p <ports>
<pre/>
--Example:
<pre>
python porter.py 192.128.2.15 -p 80,443,8080
<pre/>
